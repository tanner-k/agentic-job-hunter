# Current Project

## What we are building
We are building an agentic application pipeline that can run autonomously with a given query that contains information, such as job title, minimum salary, location, skills, keywords, and companies. Then it passes that information off to a team of agents that will go through and look for jobs, understand the job application needs, and then fill out that information, using an agent for each task based on Tanner's resume and information within the job description, and then apply automatically. We are also building a dashboard that will be able to track progress and understand what is working and not working. 

## What good looks like
Successful project for this looks like a pipeline that is able to take the query information, pass it off to the agents, and let them apply to these jobs autonomously with at least 50% success rate of application submissions. 

## What to avoid
I do not want a pipeline that sources potential jobs for me to apply to.
